# Full Stack Learning Repository (WIP)

## Overview

This repository tracks my progression through full-stack development by building projects at different stages of learning.
Each project focuses on understanding core concepts and implementation details rather than production-level polish.

---

## Current Projects

### 1. Cat Website (HTML + CSS Basics)

A foundational project focused on understanding basic HTML structure and CSS styling.

**Core Concepts Practiced:**

* Semantic HTML structure (`section`, `nav`, `form`)
* Linking external CSS 
* CSS selectors (class, id)
* Basic styling (colors, fonts, layout)
* Image embedding using external URLs
* Forms and input elements

**CSS Concepts:**

* Global styling via `body` 
* Class-based styling (`.paragraph`)
* ID-based styling (`#first-div`)
* Specificity and overrides (`!important` usage)

**Purpose:**
This project serves as an entry point into:

* how HTML is structured
* how CSS is applied and overridden
* how the browser renders basic layouts

---

### 2. Google Homepage Clone (Layout + UI Practice)

A more advanced static project replicating the layout and structure of the Google homepage.

**Core Concepts Practiced:**

* Flexbox layout system
* Responsive design using media queries
* Navigation bar structuring (left/right alignment)
* Input field styling and interaction states
* Button and icon design
* Footer layout and grouping

**UI/UX Elements:**

* Search bar with icons
* Hover and focus states
* Circular icon buttons
* Structured footer with real links

**Purpose:**
This project focuses on:

* translating a real-world UI into code
* mastering layout control using Flexbox
* understanding spacing, alignment, and responsiveness
